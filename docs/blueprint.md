# **App Name**: Auria

## Core Features:

- Lifestyle Hero Section: Display a full-width video or high-resolution slider showcasing lifestyle shots on the homepage.
- AI-Powered Gift Guide: Implement a tool that provides gift recommendations based on occasion, recipient, and budget using generative AI.
- Ring Size Assistant: Provide an interactive ring size guide to assist users in determining their ring size.
- Material Filtering: Advanced filtering on the PLP by Material (Gold, Silver, Rose Gold), Stone (Diamond, Ruby), Price Range, and Style.
- High Resolution Zoom: Enable zoom functionality for high-resolution product images to view detailed material specs on the PDP.
- Trust Signals: Prominently display icons for "Certified Authenticity," "Free Shipping," and "Lifetime Warranty" near the Add to Cart button on the PDP.
- Instagram Feed: Integrate an Instagram feed to showcase social proof and brand lifestyle.

## Style Guidelines:

- Primary color: Gold (#F4B625), symbolizing luxury and sophistication.
- Background color: Light beige (#F8F7F5), creating a soft and elegant backdrop.
- Accent color: Warm gray (#8A7E60), providing subtle contrast and a sense of understated elegance.
- Headings: 'Playfair' serif for elegance. Note: currently only Google Fonts are supported.
- Body: 'Lato' sans-serif for readability. Note: currently only Google Fonts are supported.
- Use minimalist and elegant icons to represent various features and categories.
- Employ generous whitespace to create a sense of luxury and calm, with high-quality uniform photography on gray backgrounds.
- Incorporate subtle and smooth animations for transitions and interactions to enhance the user experience.
